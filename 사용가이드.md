# 🚀 배포지원 투입관리 대시보드 사용가이드

**접속 주소:** https://emitlight.github.io/deploy-schedule/

---

## 처음 접속했을 때

페이지에 처음 들어오면 **아무 일정도 없는 빈 달력**이 보입니다. 정상입니다.  
아래 순서대로 한 번만 설정하면 이후에는 자동으로 데이터가 채워집니다.

---

## 1단계 — Worker URL 연결 (최초 1회)

> 이 설정은 처음 한 번만 하면 됩니다. 브라우저에 저장되어 이후엔 자동 적용됩니다.

1. 헤더의 **☁️ 아웃룩 동기화** 버튼 클릭
2. 팝업이 뜨면 아래 URL을 그대로 복사해서 붙여넣기 후 확인

```
https://deploycalendar.hayoung-lee.workers.dev
```

3. **아웃룩 동기화** 버튼 <u>한 번 더</u> 클릭
3. SHIELD Gate 프록시 설정으로 인해 불가한 경우, 윈도우 설정 > 프록시 설정 변경 > 설정 스크립트 사용 '끔' > (일정 동기화 완료 이후 프록시 '켬' 으로 재설정)

---

## 2단계 — 아웃룩 일정 불러오기

> 아웃룩 일정은 매일 **오전 7시 · 낮 12시 30분 · 오후 3시**에 자동으로 동기화됩니다.  
> 처음 접속했을 때는 수동으로 한 번 불러와야 합니다.

헤더의 **☁️ 아웃룩 동기화** 버튼 클릭 → 잠시 기다리면 8명의 일정이 달력에 표시됩니다.

## 3단계 — 화면 보는 법


| 색상 | 의미 |
|------|------|
| 🔵 파랑 | 회의·아웃룩 일정 |
| 🔴 빨강 | 배포지원 투입 확정 |
| ⚫ 검정 | 부재중·OOF |
| 🟤 갈색 | 기타 |

### 색상 자동 분류 기준

아웃룩 일정은 제목 키워드에 따라 자동으로 색상이 분류됩니다.

| 분류 | 적용 기준 |
|------|----------|
| ⚫ 외근/휴가 | 제목에 포함: 연차, 반차, 휴가, 외근, 출장, 재택, 공휴일, 경조, 병가, 조퇴 |
| 🔴 배포지원 | 제목에 포함: 배포, deploy |
| 🔵 회의 (예외) | 배포회의, [내부업무], [내부] 포함 시 → 배포 키워드 있어도 회의로 분류 |
| 🔵 회의 | 위 키워드 없는 모든 일정 (기본값) |

> **우선순위:** OOF(부재중) > 외근/휴가 키워드 > 회의 예외 > 배포 키워드 > 회의(기본)

### [티켓] 일정 처리 방식

제목이 `[티켓]`으로 시작하는 아웃룩 일정은 달력에 표시되지만, **라운드로빈 돌림판에서 배포 가능 시간으로 인식**합니다.

예: `[티켓] TT119590 고객사 이슈` → 해당 시간에 이 일정이 있어도 돌림판에서 제외되지 않음

티켓 처리 중에도 현장 배포지원 투입이 가능한 경우를 위한 설정입니다.

- **◀ ▶** 버튼으로 주 이동
- **요일 탭** 클릭으로 해당 날짜만 보기
- **🟢 빈 시간 보기** 버튼으로 가능한 시간대 한눈에 확인

## 배포 담당자 뽑기 — 핵심 기능!

### Step 1. 돌림판 열기

헤더의 **🎲 라운드로빈 돌림판** 버튼 클릭

### Step 2. 배포 날짜·시간 설정

```
📅 배포 일시   2026-06-10   [09:00 ▼] ~ [18:00 ▼]
```

날짜와 시간을 설정하면 **그 시간에 일정이 없는 사람만** 돌림판에 올라옵니다.  
일정이 있는 사람은 자동으로 회색 처리(취소선)됩니다.

### Step 3. 돌리기

**🎲 돌리기!** 버튼 클릭 → 당첨자 발표 🎉

> 🔄 최근에 뽑힌 사람은 자동으로 후순위로 밀립니다.  
> 공정한 순환 배정이 자동으로 유지됩니다.

### Step 4. 배포 일정 확정

당첨 후 **🚀 이 멤버로 배포지원 일정 투입하기** 클릭  
→ 자동으로 폼이 채워집니다. 고객사/프로젝트명만 입력 후 **투입 확정**  
→ 달력에 🔴 빨간색으로 표시되고 팀 전원에게 공유됩니다.

### (구현 예정) vtiger CRM 티켓 연동

당첨 직후 **🔗 vtiger CRM 티켓 연동** 섹션에서 티켓 번호 입력 → 조회 → **담당자 자동 변경** 가능

> 단, vtiger 서버(crm.softcamp.co.kr)에서 Cloudflare IP 허용이 필요합니다. 미설정 시 연동 불가.

## 일정 직접 추가

아웃룩에 없는 일정을 수동으로 등록할 수 있습니다.

1. 달력의 **빈 칸 클릭** → 일정 추가 팝업
2. 제목·시간·종류 입력 후 저장
3. 저장된 일정은 Notion에 자동 저장 → 팀 전원이 바로 확인 가능

## 자주 묻는 질문

**Q. 아웃룩 일정이 안 보여요.**  
A. **☁️ 아웃룩 동기화** 버튼을 눌러 수동으로 불러오세요.

**Q. 돌림판에 내가 안 나와요.**  
A. 설정한 배포 시간대에 내 일정이 잡혀 있으면 자동 제외됩니다. 시간대를 바꿔보세요.

**Q. 다른 사람 PC에서 일정이 다르게 보여요.**  
A. 페이지를 새로고침하면 Notion에서 최신 데이터를 불러옵니다.

**Q. Worker URL 설정을 다시 하고 싶어요.**  
A. **☁️ 아웃룩 동기화** 버튼을 클릭하면 URL 입력 팝업이 다시 뜹니다. 새 URL을 입력하세요.
